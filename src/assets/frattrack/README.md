# FratTrack screenshots

Drop the iPhone simulator screenshots here. Any number works — they are
auto-discovered by `src/data/projects.ts` and shown in the phone carousel,
ordered by the number in the filename.

Naming: `frattrack-1.png`, `frattrack-2.png`, … (`.png`, `.jpg`, `.jpeg`,
or `.webp` are all picked up). The number controls the order, so
`frattrack-10` correctly comes after `frattrack-9`.

Until at least one image is added, FratTrack falls back to the placeholder
graphic on the projects page.
